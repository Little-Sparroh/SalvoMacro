# Changelog

## 1.2.0

### Features
* Initial release (split from EnhancedCooldowns)
* Configurable wingsuit salvo activation modes
  - None: default manual firing
  - Toggle: toggle auto-fire on/off; fires automatically when recharged and enabled
* Optional suppression of the 3D salvo launcher targeting model
* HUD notification suppression during automatic salvo firing
* Live config reload with toggle state reset
